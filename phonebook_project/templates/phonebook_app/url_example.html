Примеры работы ссылок

Смотрите одновременно и на код темплейта, и на отрендеренную страницу

Есть ссылки, в которых нет параметров, после рендера будут выглядеть так:

  • {% url 'phonebook_app:main_page' %}
  • {% url 'phonebook_app:stats' %}

Есть ссылки, в которых один параметр, например:

  • {% url 'phonebook_app:list_by_city' 'Moscow' %}

Есть ссылки, в которых много параметров, например:

  • {% url 'phonebook_app:list_by_city_and_age' 'Moscow' 39 %}
  • {% url 'phonebook_app:list_by_city_and_age_range' 'Moscow' 30 45 %}

Параметры можно вставлять не только позиционно, но именованно (главное не смешивать):

  • {% url 'phonebook_app:list_by_city' gorod='Moscow' %}
  • {% url 'phonebook_app:list_by_city_and_age' city='Moscow' age=39 %}
  • {% url 'phonebook_app:list_by_city_and_age_range' city='Moscow' age_from=30 age_to=45 %}

Во всех примерах выше я использовал напрямую значения (литералы), а можно брать переменные из контекста

  • {% url 'phonebook_app:list_by_city_and_age' city=city_from_context age=age_from_context %}
  • {% url 'phonebook_app:list_by_city' gorod=some_other_dict.nested_params_example %}
